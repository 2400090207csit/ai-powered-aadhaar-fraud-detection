# AI-Powered Aadhaar Fraud Detection System

## Overview
This project detects fraudulent Aadhaar documents using OCR, computer vision, and deep learning techniques.

## Execution Environment
- Google Colab for code execution
- Python programming language
- OpenCV and Tesseract OCR
- YOLO for fraud detection

## Project Modules
1. Data Collection and Preprocessing
2. AI-OCR and Document Verification
3. Fraud Detection using Deep Learning
4. System Integration and Decision Making

## Agile Methodology
This project follows Agile methodology and includes:
- User stories
- Product backlog
- Sprint planning
- Burndown chart

## Output
The system produces a final decision:
- VERIFIED DOCUMENT
- FRAUD DOCUMENT
