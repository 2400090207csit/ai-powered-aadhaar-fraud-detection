# User Stories

1. As a verifier, I want the system to extract Aadhaar details automatically using OCR so that manual errors are reduced.

2. As a system, I want to detect document tampering so that fraudulent Aadhaar documents can be flagged.

3. As an administrator, I want a final fraud decision so that I can take appropriate action.

4. As a reviewer, I want clear output results so that document verification is easy.
